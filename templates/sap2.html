Yeni Bir Model

SAP2000'de yeni bir model oluşturmak için File sekmesinden New Model komutunu seçebileceğimiz gibi kısayol tuşu olan Ctrl+N tuşlarınıda girebiliriz. Ayrıca araç çubuğunun hemen altında bulunan yardımcı çubuk üzerinde en solda bulunan komut (New Model...) 'e de tıklayarak yeni bir model oluşturulabilir. Bu seçeneklerden herhangi biri seçilip kullanıldığında New Model penceresi ekrana gelecektir.

Burada New Model Initialization grup sekmesinde bulunan Initialize Model from Defaults with Units seçeneği modeli başlangıç ayarları ile oluşturmak için kullanılır.
Initialize Model from an Existing File seçeneği ile de daha önceden var olan bi projenin ayarları kullanılarak model başlatılmak istendiğinde kullanılabilir. Burada dikkat edilmesi gereken Initialize Model from an Existing File kullanıldığında önceki projeden oluşturulmuş malzeme tanımları, kesitler, yük ve yük kombinasyonları da beraberinde gelecektir.

Select Template grup sekmesi ise model,programda varolan şablonlar kullanılarak hızlı bir biçimde oluşturmak için kullanılabilir.

Blank ; hiçbir şablon kullanmadan model oluşturmak için kullanılabilir.
Grid Only ; sadece aks sistemi oluşturmak için kullanılabilir.
Beam ; bir veya çok açıklıklı kiriş oluşturmak için kullanılabilir.
2D Trusses ; iki boyutlu makas sistemi oluşturmak için kullanılabilir.
3D Trusses ; üç boyutlu makas sistemi oluşturmak için kullanılabilir.
2D Frames ; iki boyutlu çerçeve modeli olşturmak için kullanılabilir.
3D Frames ; üç boyutlu çerçeve modeli olşturmak için kullanılabilir.
Wall ; perde duvar modeli oluşturmak için kullanılabilir.
Flat Slab ; kirişsiz döşeme modeli oluşturmak için kullanılabilir.
Shells ; kabuk elemanlar oluşturmak için kullanılabilir.
Staircases ; merdiven modeli oluşturmak için kullanılabilir.
Storage Structures ; depo yapı modelleri oluşturmak için kullanılabilir.
Underground Concrete ; yeraltı yapı modelleri oluşturmak için kullanılabilir.
Solid Models ; katı model oluşturmak için (örn; kalın kemer) kullanılabilir.
Pipes and Plates ; boru, kanal gibi model oluşturmak için kullanılabilir.

Project Information grup sekmesindeki Modify/Show Information... butonu ile projeye ait bilgiler (şirket ismi, müşteri ismi, proje ismi, revizyon numarası vb.) girilebilir.

Şimdi Grid Only seçeneğini seçelim. Açılan Quick Grid Lines penceresinde Cartesian seçeneği kartezyen koordinatlarına göre, Cylindrical seçeneği ile silindirik koordinatlara göre aks sistemi oluşturmamıza olanak sağlar. Şimdilik kartezyen koordinatlarını kullanalım.

Burada Number of Grid Lines seçeneği ile X,Y,Z doğrultusunda aks sayısı;
Grid Spacing seçeneği ile X,Y,Z doğrultusunda aks aralıkları;
First Grid Line Location seçeneği ile X,Y,Z doğrultusunda aks sisteminin yerleşeceği koordinat noktası ayarlanabilir.

Şimdilik hiçbir ayarı değiştirmeden OK butonuna tıklayarak aks sistemimizi oluşturalım. Ekrana gelen pencerede herhangi bir noktaya mouse'la sağ tıklayarak açıklan grup sekmesinde Edit Grid Data seçeneği ile varolan aks sistemi değiştirilebilir, silinebilir, yeni bir aks sistemi oluşturulabilir.

Malzeme Özelliklerinin Tanımlanması

Projeye ait malzeme özelliklerinin programda belirlenebilmesi için Define sekmesinden Materials... komutu seçilir. Ekrana Define Material penceresi gelir. Burada Materials sekmesi altında varolan malzeme listesi gösterrilir.Click to: sekmesi altındaki;

Add New Material... seçeneği ile yeni bir malzeme tanımı yapılabilir;
Add Copy of Material... seçeneği ile varolan bir malzemenin özellikleri kopyalanabilir;
Modify/Show Material... seçeneği ile varolan malzemelerin özellikleri görüntülenebilir ve değiştirilebilir;
Delete Material... seçeneği ile varolan bir malzeme silinebilir.

Add New Material... butonuna tıklayalım. Ekrana Add Material Property penceresi açılacaktır. Burada;

Region bölümünden Europe seçilerek;
Material Type böümümünden Concrete seçilerek;
Standart bölümünden EN 1992-1-1 per EN 206-1 seçilerek;
Grade bölümünden C30/37 seçilerek C30 betonu tanımlanabilir.

Açılan Define Material penceresinde Material sekmesi altından C30/37 seçilerek Modify/Show Material... seçeneği seçilerek açılan Material Property Data penceresinden C30/37 betonuna ait malzeme özellikleri incelenebilir.

Burada;

Material Name and Display Color malzeme ismi ve bu malzemenin kullanıcı arayüzünde hangi renk ile gösterileceğini;
Weight per Unit Volume birim hacim ağırlığını;
Modulus of Elasticity, E elastisite modülünü;
Poisson, U poisson oranını;
Coefficient of Thermal Expansion termal genleşme kaysayısını;
Shear Modulus, G kayma modülünü;
Specified Concrete Compressive Strenght, f'c beton karakteristik basınç dayanımını;
Expected Concrete Compressive Strenght beton beklenen basınç dayanımını;

temsil eder.
